# Collisions in the displaying region

This interactive map visualizes traffic collisions in Seattle using data from the Seattle Open Data Portal. Circle size and color represent collision severity levels:

Severity 1: Property Damage Only
Severity 2: Injury Collision
Severity 3: Fatal Collision

The coordinated bar chart updates dynamically based on the current map view. Click on a bar to filter collisions by severity.
